# Discord Bot

## Author Ammar Abul-Feilat

## **Version 0.1.0**

## overview

A basic Bot for discord.

!info -> get info about the bot
!ping -> ping
!hello -> hello
!roll -> roll a dice
!flip -> flips a coin
!exit -> gracefully exits the bot
!choose -> rock, paper, scissors
!time -> get the current time
!help -> get help
